# Epic 03: 薪酬基础（Payroll Basic）

**对标**：SAP PY / PeopleSoft Payroll（严格简化版）

## 业务目标
每月为在职员工执行一次性工资计算 run，输出工资条；支持锁定与红冲。个税与社保公积金使用单一示例口径。

## Sprint 3 Stories

| Story | 名称 | 范围 | 状态 |
|---|---|---|---|
| EP03-S09 | 薪资档案 + 工资计算表结构 | 基础设施层 | pending |
| EP03-S10 | 工资 Run 计算引擎 + 红冲 + 工资条 | 核心计算层 | pending |

## 硬边界
- 不做按日折算
- 不做跨月补发补扣
- 不做专项附加扣除
- 不做年度汇算
- 不做多套薪资方案
- 不做考勤/加班联动
- 不允许直接编辑已锁定 run，只能红冲重算
- 入职/离职当月员工不参与 run
